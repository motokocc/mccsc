# xlord

#### Description
motoko多canister框架
由center，service，controller组成，三个容器都可以单独部署
center管理用户，存储用户数据，所有的业务都从这个服务创建，然后交给controller管理
service主要处理业务，只关注业务逻辑的实现，不需要考虑存储
controller 管理业务，存储业务相关数据，
